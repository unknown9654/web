# 家书

这是已经适配 GitHub Pages 的完整网页项目。无论仓库叫什么名字，自动部署时都会处理正确的资源路径。

## 上传与发布

1. 在 GitHub 新建一个公开仓库。
2. 解压桌面上的压缩包，把解压后的全部内容上传到仓库根目录并提交。
3. 打开仓库的 `Settings → Pages`。
4. 在 `Build and deployment` 中，把 `Source` 选择为 `GitHub Actions`。
5. 打开仓库的 `Actions` 页面，等待“部署前进民爆安全家书”显示绿色对勾。
6. 回到 `Settings → Pages`，复制页面显示的网址即可分享。

> 网页已包含手机端滑动、点击互动、液态玻璃动效、前小进形象和提示音。首次轻触“开启家书”后，浏览器才会播放声音。
